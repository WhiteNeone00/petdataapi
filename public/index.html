Pet Simulator 99 Public API

Access in-game data for pets, items, clans, and more

Introduction

The Pet Simulator 99 Public API provides access to in-game data and configuration details. With this API, developers can retrieve information about pets, items, clans, and more, allowing for the creation of external tools, websites, Discord bots, and other applications.

üìç Production URL

https://yourapi.example.com/api/

Replace with your actual Firebase Hosting URL

API Endpoints

GET /api/collections

Types of data from Pet Simulator 99's in-game configuration. These are like tables of a database.

Request
curl 'https://yourapi.example.com/api/collections'
Response
{
  "status": "ok",
  "data": [
    "Achievements", "Boosts", "Booths", "Boxes",
    "Buffs", "Charms", "Currency", "Eggs",
    "Enchants", "FishingRods", "Fruits", ...
  ]
}
GET /api/collection/{collectionName}

Details from a specified collection. Contains configuration data like rows of a database.

Request
curl 'https://yourapi.example.com/api/collection/Pets'
Response
{
  "status": "ok",
  "data": [
    {
      "configName": "Huge Cosmic Axolotl",
      "category": "Huge",
      "configData": {
        "name": "Huge Cosmic Axolotl",
        "thumbnail": "rbxassetid://15201636161"
      },
      "collection": "Pets"
    }
  ]
}
GET /api/clansList

Get a list of all clan names.

Request
curl 'https://yourapi.example.com/api/clansList'
Response
{
  "status": "ok",
  "data": ["KIL1", "ukbd", "c1aw", "hdkn", "C0ME", ...]
}
GET /api/clansTotal

Get the total number of clans.

Request
curl 'https://yourapi.example.com/api/clansTotal'
Response
{
  "status": "ok",
  "data": 145463
}
GET /api/clans

Get paginated list of clans with sorting options.

Query Parameters:
  • page (optional): Page number (default: 1)
  • pageSize (optional): Items per page (default: 10)
  • sort (optional): Field to sort by (default: Points)
  • sortOrder (optional): "asc" or "desc" (default: desc)
Request
curl 'https://yourapi.example.com/api/clans?page=1&pageSize=10&sort=Points&sortOrder=desc'
GET /api/clan/{clanName}

Get details for a specific clan.

Request
curl 'https://yourapi.example.com/api/clan/CAT'
GET /api/exists

Exists data for each item and pet in the game.

Notes:
  • For pets: pt=1 is golden, pt=2 is rainbow, sh is shiny
  • For potions/enchants: tn is tier number
GET /api/rap

Get RAP (Recent Average Price) data for items.

Request
curl 'https://yourapi.example.com/api/rap'
GET /api/activeClanBattle

Get information about the currently active clan battle.

GET /image/{imageId}

Displays a Roblox library asset as an image by proxying the request.

Request
curl 'https://yourapi.example.com/image/14615650278'

API Information

üìã Standard Responses

Success (200): Returns JSON with status: "ok" and optional data field.

Error (400): Returns JSON with status: "error", error message, and ignore flag.
